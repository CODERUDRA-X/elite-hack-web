## 🌟 About The Project
This is a futuristic, high-performance landing page designed for a Hackathon. It features a "Deep Space Cyberpunk" theme with neon aesthetics, focusing on speed, responsiveness, and visual impact.

Built with the goal of creating an **Awwwards-level** user interface.

### ✨ Key Features
* **🌌 Deep Space Theme:** Custom starfield background with purple/pink neon glows.
* **🚂 Infinite Logo Scroll:** A smooth, auto-scrolling marquee for Sponsors.
* **⚡ Modern Tech Stack:** Built using the latest Next.js 15 & React 19.
* **🎨 Glassmorphism:** Stylish translucent cards for the Timeline and Prizes.
* **📱 Fully Responsive:** Looks great on Mobile, Tablet, and Desktop.

---

## 🛠️ Tech Stack
* **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Tailwind Animate](https://github.com/jamiebuilds/tailwindcss-animate)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Font:** Orbitron (Headings) & Space Grotesk (Body)
* **Deployment:** Vercel

## 🚀 How to Run Locally

If you want to check out the code or run it on your machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/elite-hack-web.git](https://github.com/your-username/elite-hack-web.git)
    cd elite-hack-web
    ```

2.  **Install Dependencies:**
    ```bash
    npm install --legacy-peer-deps
    ```

3.  **Run the Server:**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser.
